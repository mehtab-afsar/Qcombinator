# Phase 2: Agents That Connect to Real Tools

> After the core agent deliverable system is built, Phase 2 makes outputs actionable by pushing them directly into the tools founders already use.

---

## GTM Agent (Patel)
- [x] Generates a **Carrd/Framer landing page** you can publish in one click (Download HTML button → self-contained landing page file)

## Sales Agent (Susi)
- [x] Pushes outreach sequences directly to **Instantly / Smartlead / Apollo** (Gmail compose links on each email step)

## Finance Agent (Felix)
- [x] Exports the financial model to **Google Sheets** with formulas intact (CSV with formula rows for MRR growth, churn, net cash — import via File → Import → Upload)

## PMF Agent (Nova)
- [x] Creates a **Typeform / Tally survey** you can send immediately — no copy-pasting (Download Survey HTML → self-contained form with Ellis test + follow-ups, stores responses in localStorage)

## Brand Agent (Maya)
- [x] Generates **Figma-ready social media templates** — drop into Figma and customise (Download HTML → 3 SVG templates: Instagram 1080×1080, Twitter card 1200×628, LinkedIn banner 1584×396)

## HR Agent (Harper)
- [x] Publishes job descriptions as drafts to **Wellfound / LinkedIn** (one-click post + copy JD per role)

## Legal Agent (Leo)
- [x] Pre-fills a **Clerky / Stripe Atlas** incorporation flow from collected data (Copy Details button pre-loads company info + one-click launch on Clerky or Stripe Atlas)

## Competitive Intel Agent (Atlas)
- [x] Sets up **Google Alerts** for every competitor automatically

## Strategy Agent (Sage)
- [x] Pushes OKRs to **Linear / Notion** as trackable goals with owners and due dates (Export button copies markdown OKRs + opens Linear)
